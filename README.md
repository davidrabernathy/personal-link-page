# Personal Link Page

a clean, modern landing page showcasing my links and projects

## Live Demo

**https://davidrabernathy.github.io/personal-link-page/

## Features

- Modern design with blurred background image
- SVG icons for professional, scalable graphics
- Smooth hover animations and transitions
- Single-card layout for clean, focused user experience
- Fully responsive design across all devices (mobile, tablet, desktop)
- Optimized performance with lightweight code

## Technologies
- **HTML5** - Semantic markup and structure
- **CSS3** - Styling techniques
 - Flexbox for layout
 - CSS transformations and transitions for animations
 - Media queries for responsive design
- **SVG Icons** - Scalable vector graphics
- **GitHub Pages** - Static site hosting

## Project Structure
```
link-page/

├── style.css            # All styling and animations
├── *.svg                # SVG icon files
├── resume.pdf           # Downloadable resume

```

### Key Concepts Demonstrated

- **Modern CSS Techniques**: Blurred background
- **Responsive Design**: Mobile-first approach with flexible layouts
- **SVG Integration**: Using vector graphics for crisp icons at any size
- **User Experience**: Subtle animations and hover states for interactivty
- **Clean Code**: Separation of concerns with organized file structure
- **Deployment**: Git version control and live deployment with GitHub Pages
